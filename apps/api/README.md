# Backend API

Server-side API for the AI Image Platform.

## Structure

- `src/modules/` - AI module endpoints
- `src/prompt_brain/` - Prompt versioning
- `src/batch/` - Batch processing
- `src/jobs/` - Job management
- `src/assets/` - Asset handling
- `src/auth/` - Authentication
- `src/db/` - Database layer
